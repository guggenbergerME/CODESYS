# BoschRexroth Software

## Installation WORKS
Die aktuelle Version kann unter myRexroth im Bereich ```myRexroth | Collaboration | Collaboration Rooms``` heruntergeladen werden. Zum Betrieb wird ein Windows 11 benötigt.
